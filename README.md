## ✨ 设计思想

软件设计的目标群体为跟作者一样上水课的大学牲，英语学习不仅是掌握一门语言，更是打开全球机遇的钥匙。点滴学习。

**“水课可以水，但你的未来不能水。”**

现在开始，每天记一点，让英语成为你**最硬核的摸鱼技能** 💪
